# Battery-surface-defect-detection

一个基于深度学习的电池表面缺陷检测系统，支持自动识别电池表面的划痕、凹陷、污渍等常见缺陷。

项目亮点

- 🚀 使用 YOLOv8 实现高精度缺陷检测
- 📊 支持多种工业电池类型
- 🔍 提供可视化界面展示缺陷位置和置信度
- ⚡ 优化推理速度，满足实时检测需求
